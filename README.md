# SuperList

This is a regular groceries/todo list, developed for personal uses and as playground for learning new technologies

## Installation

```bash
yarn install
```

## Processes

```bash
# uses a dev db.
yarn dev

# vite build
yarn build

# vite preview
yarn preview

# vitest watch session
yarn test

# coverage (c8 + instanbul)
yarn coverage

# storybook
yarn sb

# storybook build
yarn build-storybook
```
